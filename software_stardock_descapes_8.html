Installation

  • Télécharger le logiciel.
  • Exécuter "Setup.exe".
  • Poursuivre l'installation (Peut prendre plusieurs minutes)
  • Lancer Stardock Descapes 8
  • Cliquer sur "Start 30 Days Trial"
  • Mettre une adresse mail quelconque (Avec consultation des mails possibles)
  • Valider l'activation via le mail reçu
  • Lorsque la période d'essai expire :
  • Ouvrir la rechercher Windows avec les touches "Windows + R" simultanément.
  • Se rendre dans "%programdata%
  • Ouvrir le dossier "Stardock"
  • Ouvrir le dossier "Deskcapes"
  • Supprimer les fichiers "Licence.sig" et "Cache.dat"
  • Répéter la procédure.
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
