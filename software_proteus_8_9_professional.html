Installation

  • Télécharger le logiciel.
  • Extraire l'archive "Proteus Professional 8.9 SP0 Build 27865".
  • Ouvrir le dossier "Proteus Professional 8.9 SP0 Build 27865".
  • Exécuter P8.9SP0_Setup.exe".
  • Choisir lors de l'installation : "Use a locally installed license key".
  • Au message qui dit "No license Key found for Proteus", cliquer sur "Next".
  • Cliquer sur "Browse For Key File".
  • Rechercher la licence. (Qui se trouve dans le dossier "crack")
  • Une fois la licence importé dans l'installation, cliquer sur "Install" puis sur "Close".
  • Poursuivre et attendre l'installation du logiciel.
  • Retourner dans le dossier "Crack".
  • Copier le fichiers "PP8.9.exe" + "Licence.lxk".
  • Coller les fichiers dans le répertoire de Proteus. (Programmes (x86)/Labcenter Electronics/Proteus 8 Professional/BIN)
  • Exécuter "PP8.9.exe".
  • Lancer Proteus 8.
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
