Installation

  • Télécharger le logiciel.
  • Extraire l'archive "WeMod PRO"
  • Exécuter "WeMod Updater.exe" (Peut prendre plusieurs minutes)
  • Poursuivre l'installation.
  • Pré-créér un compte WeMod
  • Lancer WeMod.
  • Il est possible qu'au premier lancement, l'application semble fixe, dans ce cas relancer l'application.
  • ⚠️ Ne pas changer son profil (Nom / Mot de passe / Photo) une fois connecté.
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
