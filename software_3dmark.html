Installation

  • Procédure d'installation :
  • IMPORTANT - Désactiver votre antivirus
  • Télécharger le logiciel. (Avec qBittorrent)
  • Extraire "Futuremark 3DMark Professional....zip" (Mot de passe : 123)
  • Ouvrir le dossier "setup".
  • Exécuter "3dmark-setup.exe".
  • Attendre la fin de l'installation.
  • Ouvrir le dossier "keygen".
  • Exécuter "kg.exe".
  • Générer une clé produit.
  • Copier la clé produit.
  • Activer 3DMark.
  • Lancer 3DMark
  • Se rendre dans l'onglet "Options"
  • Décocher la case "Valider le résultat en ligne" (Si disponible)
  • Dans "License", coller la clé produit.
  • ⚠️ Si "licence invalide", bloquer 3DMark dans le pare-feu, et recommencer l'opération.
  • Redémarrer 3DMark
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
