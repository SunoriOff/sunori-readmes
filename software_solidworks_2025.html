Installation

  • Télécharger le logiciel (avec qBittorrent)
  • IMPORTANT - Couper la connexion Internet
  • Vérifier que les fonctionnalités Windows ".NET Framework 3.5 et 4.0" sont installé sur le PC
  • Accéder au panneau de configuration → Programmes → Activer ou désactiver des fonctionnalités Windows → Cocher ".NET Framework 3.5"
  • (Si installé) Désinstaller "SolidWorks_Flexnet_Server"
  • Ouvrir le dossier "Crack"
  • Ouvrir le dossier "_SolidSQUAD_"
  • Ouvrir le dossier "SolidWorks_Flexnet_Server"
  • Exécuter "server_remove.bat" (En tant qu'administrateur)
  • Ajouter la clé de registre
  • Ouvrir le dossier "Crack"
  • Ouvrir le dossier "_SolidSQUAD_"
  • Installer la clé de registre via "sw2025_network_serials_licensing.reg"
  • Installer les services "Flexnet_Server"
  • Copier/Coller le dossier "SolidWorks_Flexnet_Server" dans le disque système. (Généralement C:)
  • Ouvrir le dossier "SolidWorks_Flexnet_Server".
  • Exécuter "server_install.bat" (En tant qu'administrateur)
  • Attendre que la console affiche "le service a démarré"
  • Installer SolidWorks"
  • Ouvrir le dossier "Setup"
  • Exécuter "Setup.exe"
  • RESERVE "Installer les composants du serveur". (Si sélectionner "Installer sur cet ordinateur", ignorer ces deux étapes.)
  • Si sélectionner "Installer les composants du serveur" → "SOLIDWORKS PDM Server" → NE PAS INSTALLER "SolidNetwork License Manager"
  • Lorsque l'installation demande la licence : 25734@localhost
  • Si la fenêtre d'avertissement de vérification du système apparaît, l'ignorer.
  • Si l'avertissement « Le numéro de série SolidWorks n'a pas été trouvé dans la base de données d'activation » apparaît, l'ignorer.
  • Si la liste complète des produits SolidWorks à installer n'est pas visible :
  • Cliquer sur "Sélectionner un forfait différent"
  • Cocher l'option "Sélectionner des produits non inclus dans ce forfait"
  • Sélectionner les produits SolidWorks à installer. (SAUF SW Manage Client & SW PDM Client)
  • Si le message « Installation Manager n'a pas pu déterminer l'abonnement en cours car il a expiré.
  • Souhaitez-vous réactiver votre licence pour mettre à jour ces informations ?" apparaît :
  • Cliquer sur Non
  • Cliquer sur Oui dans « Voulez-vous le faire plus tard ? »
  • Attendre la fin de l'installation. (Peut être très long !)
  • Application du crack (Après installation terminée)
  • Ouvrir le Gestionnaires des Tâches Windows
  • Accéder aux détails
  • (Si il est présent) Fermer le processus "sldworks_fs.exe"
  • (Si il est présent) Supprimer "netapi32.dll" dans le répertoire de SolidWorks (Par défaut : C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS)
  • (Si il est présent) Renommer "StoredProcs.dll" en "StoredProcs.dll.bak"
  • Réouvrir le dossier "Crack"
  • Ouvrir le dossier "SOLIDWORKS Corp"
  • Copier tout le contenu du dossier
  • Coller le tout dans le répertoire de SolidWorks (Par défaut : C:\Program Files\SOLIDWORKS Corp\SOLIDWORKS)
  • Ajouter une nouvelle clé de registre
  • Ouvrir le dossier "Crack"
  • Ouvrir le dossier "_SolidSQUAD_"
  • Installer la clé de registre via "SolidSQUADLoaderEnabler.reg"
  • Redémarrer votre PC
  • Lancer SolidWorks 2025.
  • Aller dans "Aide" → "SolidNetWork License Manager" → "Ordres des licences"
  • Utilisez les boutons « Monter » et « Descendre » pour positionner les produits Premium au dessus des produits Professionnels et Standards.
  • Cliquer sur "Appliquer" pour valider les changements.
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
