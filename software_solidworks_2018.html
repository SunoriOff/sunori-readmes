Installation

  • IMPORTANT - Désactiver votre antivirus
  • Télécharger le logiciel (avec qBittorrent)
  • Ouvrir le dossier "_SolidSQUAD_"
  • Exécuter "SolidWorks 2017-2018 Activator SSQ.exe"
  • Cliquer sur "Set Serial Numbers"
  • Dans le menu déroulant, sélectionner "FORCE LOCAL ACTIVATION SERIAL NUMBERS"
  • Cliquer sur "Accept Serial Numbers" (Si message : faire "yes")
  • Cliquer sur "Activate licenses"
  • Dans le menu déroulant, sélectionner "SOLIDWORKS 2018 PRODUCTS"
  • Cliquer sur "Activate licenses" et attendre ~15 secondes (Si message : "yes")
  • Enregistrer le fichier généré sur le bureau
  • Ouvrir "SolidWorks 2018 SP0 Premium DVD"
  • Exécuter "Setup.exe"
  • A la page "Numéro de série" IMPORTANT - Couper votre connexion Internet
  • Au message d'erreur, cliquer sur "Annuler"
  • A la page "Résumé", dans options de téléchargement, désactiver "Utiliser le téléchargement en arrière plan pour les service packs à venir"
  • Retourner dans le dossier "_SolidSQUAD_"
  • Exécuter à nouveau "SolidWorks 2017-2018 Activator SSQ.exe"
  • Cliquer sur "Copy Loaders"
  • Cliquer sur "Copy Loaders" (Si message : faire "yes")
  • Cliquer sur "Activate Xpress Products"
  • Cliquer sur "Activate Express Products - RUN FOR EVERY USER" (Si message : faire "yes")
  • Cliquer sur "Activate Other Products"
  • Cliquer sur "Activate Miscellany Products" - RUN FOR EVERY USER" (Si message : faire "yes")
  • SolidWorks est activé !
  • Lancer SolidWorks
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
