Installation

  • Télécharger le logiciel.
  • Extraire l'archive "Mirillis.Action!.rar"
  • Ouvrir le dossier "Setup".
  • Exécuter "Action_setup.exe".
  • Poursuivre l'installation.
  • Ouvrir le dossier "Loader".
  • Copier le "Action_Loader.exe".
  • Coller "Action_Loader.exe" dans : Disque C​ - Program Files - Mirillis - Action!
  • Lancer Action! Mirillis. (Via Action_Loader.exe)
⚠️ Assurez-vous que votre système respecte la configuration minimale requise.
